# Create environment  
```bash  
python3 -m venv my_env  
```  
  
# Activate environment  
```bash  
# linux, mac  
source my_env/bin/activate   
# windows  
my_env/bin/activate.bat   
```  
  
# (Optional) Deactivate environment  
```bash  
deactivate   
```  
  
# Install Pandas package  
```bash  
python3 -m pip install --upgrade pip  
pip install matplotlib  
```
